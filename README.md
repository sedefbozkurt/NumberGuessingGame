# NumberGuessingGame

Bilgisayarın rastgele bir sayı seçtiği ve
oyuncunun sınırlı sayıda denemeyle bu sayıyı tahmin etmeye çalıştığı bir sayı tahmin oyunudur.

## Özellikler

- Üç farklı zorluk seviyesi
- Puan sistemi: Daha az denemeyle doğru tahmini yaparak yüksek puan kazanın!

## Nasıl Oynanır?

1. Üç farklı zorluk seviyesinden birini seçin:
   - Kolay (1 - 50)
   - Orta (1 - 100)
   - Zor (1 - 200)
2. Tahmininizi girin.
3. Verilen ipuçlarına göre sayıyı tahmin etmeye çalışın:

İyi eğlenceler!
